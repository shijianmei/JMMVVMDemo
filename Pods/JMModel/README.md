JMModel
-------------

A fast, convenient and nonintrusive conversion between JSON and model.

转换速度快、使用简单方便的字典转模型框架
> **功能:**

> - JSON --> Model
> - Model --> JSON
> - JSON Array --> Model Array
> - Model Array --> JSON Array
> - print all properties in one line code
> - model copy in one line code

---------
#### **Installation【安装】**
From CocoaPods
```
pod 'JMModel'
```
Manually
```
JMModel.h
JMModel.m
```
